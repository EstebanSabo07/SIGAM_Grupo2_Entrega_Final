Notebooks y Artefactos
======================

El repositorio incluye notebooks de replicacion y analisis que documentan el
trabajo exploratorio del proyecto:

* ``igsm_replicacion_IGSM2025.ipynb``
* ``igsm_replicacion_IGSM2025-actualizado.ipynb``

Estos notebooks se mantienen como artefactos de proyecto. No se convierten, no
se ejecutan y no participan en el build de Sphinx. Esta decision mantiene la
documentacion ligera y evita que el build dependa de kernels, salidas pesadas o
estado interactivo.

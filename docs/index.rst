Documentacion de SIGAM
======================

SIGAM, Sistema Integrado de Gestion y Analisis Municipal, es una plataforma
Streamlit para digitalizar, calcular y analizar el Indice de Gestion de
Servicios Municipales de Costa Rica.

Esta documentacion describe el proyecto en espanol. La referencia de API se
genera desde docstrings en ingles y formato Google dentro del codigo Python.

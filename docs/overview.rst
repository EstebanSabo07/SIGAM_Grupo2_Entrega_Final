Resumen del Proyecto
====================

SIGAM apoya la gestion del IGSM 2025 con una aplicacion web para dos perfiles:
municipalidades y Contraloria. Las municipalidades pueden completar el
formulario, guardar respuestas y consultar resultados. La Contraloria puede
revisar rankings nacionales, analisis por region, pesos del indice y reportes.

Objetivos principales
---------------------

* Centralizar la captura de respuestas del formulario IGSM.
* Calcular puntajes y niveles de madurez con la metodologia del proyecto.
* Consultar resultados municipales y nacionales desde una interfaz Streamlit.
* Exportar datos y reportes para analisis posterior.
* Mantener una capa ORM que permita trabajar con SQLite en desarrollo y con una
  base compatible con SQLAlchemy en otros entornos.

Alcance documentado
-------------------

La documentacion cubre los modulos Python del proyecto, la estructura general
de la aplicacion, la capa de datos y los artefactos analiticos incluidos en el
repositorio. Los notebooks se referencian como artefactos y no se ejecutan como
parte del build de Sphinx.
